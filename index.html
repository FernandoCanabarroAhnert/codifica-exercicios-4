Sobre mim

Estou atualmente no meu 4º semestre de Ciência da Computação, na Uniritter – FAPA. Gosto muito da área de tecnologia, principalmente na área de Desenvolvimento de Software e Desenvolvimento Web Full Stack. Comecei a estudar programação no início de 2024, um pouco antes de entrar na faculdade, lendo um livro sobre Lógica de Programação. Ao entrar na faculdade, eu fui introduzido ao Java, e desde então tenho me aprofundado nessa linguagem, aprendendo sobre Orientação a Objetos, Estruturas de Dados e Algoritmos, além de ter aprendido sobre o paradigma funcional. Depois de ter construído uma base sólida em Java, comecei a estudar sobre Spring Boot, a fim de desenvolver APIs RESTful e aplicações web. Com isso, tive contato com conceitos como Injeção de Dependência, Padrões de Projeto, Testes Automatizados, e boas práticas de desenvolvimento, além de ter aprendido sobre bancos de dados relacionais e não relacionais. Com um pouco mais de prática, comecei a utilizar o Docker, usando-o para instanciar containers de bancos de dados e aplicações, e também para criar imagens de aplicações com Dockerfile, facilitando o processo de deploy. E logo após eu fui saber mais sobre o conceito de CI/CD e como aplicá-lo em um projeto com Spring Boot, e comecei a utilizar o Github Actions para automatizar o processo de deploy.

Eu fiquei meses a fio somente estudando sobre back-end, até que decidi que era hora de aprender sobre front-end. Comecei a estudar HTML, CSS e JavaScript, e logo depois fui apresentado ao TypeScript. Com isso, aprendi sobre os conceitos de Responsividade, Acessibilidade, SEO, e boas práticas de desenvolvimento. Depois disso, comecei a estudar sobre frameworks de CSS, como Bootstrap e Tailwind, e pré-processadores como SASS. Logo depois, fui apresentado ao Angular, e comecei a desenvolver aplicações web com esse framework, aprendendo sobre Componentes, Serviços, Injeção de Dependência, Rotas, e boas práticas de desenvolvimento. E com isso, eu agora consigo desenvolver aplicações web Full Stack, utilizando Java e Spring Boot no back-end, e HTML, CSS, JavaScript e Angular no front-end. O próximo passo neste programa da Codifica +praTi é me aprofundar ainda mais com Java e Spring Boot e aprender a utilizar o React, a fim de me tornar um desenvolvedor Full Stack ainda mais completo.

Habilidades

No back-end, eu tenho experiência com Java e Spring Boot, além de ter prática com JavaScript e TypeScript nas ferramentas Node.js, Express.js e Nest.js. Os bancos de dados que tenho mais contato são o MySQL, PostgreSQL e MongoDB. Tenho prática com Testes Automatizados, utilizando ferramentas como JUnit, Mockito, Jest e Supertest, que incluem Testes Unitários e Testes de Integração. Para os Testes de Integração eu costumo utilizar os Test Containers do Docker. Para containerização de aplicações, eu tenho experiência com Docker e Docker Compose, e uma noção superficial de Kubernetes. Para a automação de deploys, eu utilizo o Github Actions.

No front-end, eu tenho experiência com HTML, CSS, JavaScript e TypeScript, pré-processadores como SASS, frameworks de CSS como Bootstrap e Tailwind, e com o framework Angular. Tenho prática também com Jest, para escrever Testes Unitários

Além disso, tenho vivência com Metodologias Ágeis, como Scrum e Kanban, e com ferramentas de controle de versão como Git e Github. Estou sempre buscando aprender mais e me aprimorar e evoluir constantemente, e atualmente estou estudando sobre Clean Code, Clean Architecture e SOLID.

Projetos

E-Commerce Full Stack

Um projeto de e-commerce no qual há o fluxo completo de cadastro e login, visualização e filtro de produtos, adição de produtos ao carrinho e à lista de desejos, e a finalização de compras. Há também uma área administrativa, onde é possível gerenciar produtos, categorias, usuários, cupons de desconto e pedidos, bem como gerar relatórios em formato PDF.

Java Spring Boot Docker PostgreSQL AWS HTML SASS JavaScript Bootstrap Thymeleaf

Todo App

Um projeto de Todo App, que foi o meu primeiro projeto full stack com Spring Boot e Angular. É um simples CRUD de tarefas, onde é possível criar, ler, atualizar e excluir tarefas. O diferencial desse projeto foram os testes de integração com TestContainers e o deploy de ambas as stacks na nuvem do OCI e com tráfego HTTPS.

Java Spring Boot Docker MongoDB OCI HTML SCSS TypeScript Tailwind Angular

Booking App

Projeto de site de gerenciamento de reservas com login, cadastro, recuperação de senha e visualização de hotéis e quartos com filtros. Permite criar reservas com 4 formas de pagamento (Dinheiro, Pix, Cartão e Boleto com PDF), envio de e-mails de confirmação e boleto, gestão de dados do usuário, cartões e avaliações. Possui área de gerenciamento com CRUD completo, dashboard com estatísticas e geração de relatórios (PDF/Excel), com acessos diferenciados para administrador, gerente e funcionário.

Java Spring Boot Docker PostgreSQL OCI HTML SCSS TypeScript Tailwind Angular
